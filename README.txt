INTRODUCTION
------------

Browse a public TV show API!. This is a somewhat contrived example module, but demonstrates some core Drupal and OOP
concepts.


REQUIREMENTS
------------

* PHP 7.4

INSTALLATION
------------

* Install the module

USAGE
-----

Go to /tvmaze/search, type the name of a TV show, and browse the list. Click on an individual title to see a description
and a list of cast.
